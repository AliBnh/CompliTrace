# CompliTrace Frontend

React + Vite + Tailwind SPA with 4 views:
- Upload
- Sections Review
- Findings
- Report

## Run locally

```bash
npm install
npm run dev
```

Environment variables:
- `VITE_INGESTION_URL` (default `http://localhost:8001`)
- `VITE_ORCHESTRATION_URL` (default `http://localhost:8003`)
